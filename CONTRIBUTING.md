# Guia de Contribuição

Obrigado por seu interesse em contribuir com o projeto de **Análise Comparativa de Algoritmos**! Siga as etapas abaixo para colaborar.

## 🧾 O que está neste repositório?
* Implementações de Selection Sort e Quick Sort.
* Scripts de medição de tempo e geração de números.
* Ferramentas de plotagem de gráficos comparativos.

## 📌 Requisitos para contribuir
* Conhecimento básico em **Lógica de Programação** e Algoritmos.
* Ambiente configurado com a linguagem do projeto (Python/C).
* [Git](https://git-scm.com/) instalado.

## 🚀 Etapas para contribuir
1. **Fork:** Faça um fork do projeto para sua conta.
2. **Clone:** `git clone https://github.com/Mtheusgs/MatheusGarcia_LuccasVieira_ws_AA_RR_2026.git`
3. **Branch:** Crie uma branch para sua alteração: `git checkout -b melhoria-algoritmo`
4. **Implemente:** Mantenha o código limpo, comentado e realize testes de tempo.
5. **Commit:** `git commit -m "Otimiza pivo do Quick Sort para melhor performance"`
6. **Push:** `git push origin melhoria-algoritmo`
7. **PR:** Abra um Pull Request descrevendo suas melhorias.

## ✅ Boas práticas
* **Não suba arquivos pesados:** Evite incluir arquivos de dados (.txt, .csv) gigantes; prefira subir o script que os gera.
* **Limpeza:** Não envie pastas de ambiente virtual (`venv/`, `__pycache__`).
* **Testes:** Garanta que o código rode corretamente em diferentes tamanhos de `range`.

## 🛠 Sugestões de contribuição
* Implementação de novos casos de teste (pior caso, caso médio).
* Melhorias visuais nos gráficos gerados.
* Otimizações de memória nas implementações dos algoritmos.
